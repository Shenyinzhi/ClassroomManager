📋 功能概述
ClassroomManager 是一套完整的教室计算机管理解决方案，旨在为教育环境提供全面的计算机使用控制。系统包含以下模块：
1. 网络访问控制
• 自动屏蔽3000+个娱乐、游戏和社交媒体网站
• 实时监控并保护hosts文件不被修改
• 自动清理浏览器历史记录和缓存
2. 外设安全管理
• 实时监控所有可移动存储设备
• 检测到娱乐内容时自动弹出U盘
• U盘密码保护系统（密码：pyas）
3. 进程监控系统
• 实时检测并终止100+个音视频娱乐软件进程
• 防止学生安装或运行未经授权的应用程序
4. 桌面环境锁定
• 强制锁定桌面壁纸为指定图片
• 防止学生更改桌面设置
• 自动恢复被修改的系统主题设置
🛠️ 系统要求
• 操作系统: Windows 7/8/10/11
• 权限要求: 管理员权限
• 硬盘空间: 至少50MB可用空间
• 运行环境: 需要启用批处理和VBScript执行权限
📥 安装与配置
快速安装：
1. 下载最新版本的ClassroomManager
2. 运行Setup.exe,为防止学生发现，安装程序无窗口，运行程序后安装完成。
手动安装：
# 克隆仓库
git clone https://github.com/yourusername/ClassroomManager.git
# 进入目录
cd ClassroomManager
# 以管理员权限运行安装脚本
运行Setup.exe
🔧 使用说明
启动系统：
• 系统安装后重启电脑将自动启动
• 手动启动：以管理员身份运行隐藏文件 “C:\Program Files\Program.exe”
U盘密码功能：
当检测到U盘中含有娱乐内容时：
1. U盘将自动弹出
2. 系统提示输入密码
3. 输入密码 pyas 可临时解除限制（仅限本次会话）
恢复工具：
系统包含完整的恢复工具 RecoveryTools.exe，可一键恢复所有修改：
重要提示：U盘被自动弹出后，需要通过以下方式重新分配盘符：
1. 打开"磁盘管理"（Win+X → 磁盘管理）
2. 找到被弹出的U盘
3. 右键点击 → "更改驱动器号和路径"
4. 点击"添加" → 分配新盘符
🚨 已知限制与注意事项
技术限制：
1. U盘弹出问题：某些U盘可能无法正常弹出，需要手动重新分配盘符
2. 进程绕过：重命名的可执行文件可能绕过检测
3. 网络代理：学生可能通过代理服务器绕过网站屏蔽
4. 安全模式：在安全模式下系统保护无效

删除“C:\Program Files\Program.exe”并重启
